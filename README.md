Share link: https://bulletin.unilife.cc/hualienCounty/activity/3sf5pJpOe20PV1VzCYsS

## Setup
```
firebase init
yarn
```
## Build
```
yarn build
```

## Deployment
```
yarn deploy
```

## Development
```
yarn dev
```
This will serve hot-reload website at `localhost:3000`
